# Digimemo

Digimemo is a single-player memory game built with React. The player watches a short sequence on a number keypad, then repeats the same sequence from memory. Each correct round adds one more number, so the game becomes more challenging over time.

https://digimemo.netlify.app/

This project was created for SEG 3125 Assignment 3 as a high-fidelity interactive prototype for a memory game.

## Features

- Three difficulty levels: Beginner, Intermediate, and Advanced
- Different game lengths: 4, 8, or 12 rounds
- Score and best score tracking
- Three lives per game
- Audio feedback with a sound on/off option
- Keyboard support for number input
- Home button to leave a game and return to setup
- Clear end screen with score, best score, and progress
